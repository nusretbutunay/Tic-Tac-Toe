# Ruby Enumarables Project

In this project, we implemented our own version of Ruby’s enumerable methods. Methods are:

#my_each
#my_each_with_index
#my_select
#my_all?
#my_any?
#my_none?
#my_map
#my_inject
#my_count
#my_multiply_els (to test inject)

## Built With

- Ruby
